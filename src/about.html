Syfte och information

Den här webbplatsen läser ut poster från min REST-webbtjänst som jag skapade med MongoDB Atlas. Denna webbapplikation som är frontend-delen av projektet använder Fetch API för att hämta, skicka data samt radera från och till webbtjänsten.

Slutsats om uppgiften

Kul att få testa att återanvända samma frontend-applikation med en annan webbtjänst. Jag tyckte det var väldigt smidigt att använda MongoDB Atlas och att implementera den till denna webbapplikation gick smidigare än vad jag trodde.
